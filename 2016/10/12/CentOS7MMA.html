CentOS 7 使用McMyAdmin搭建Minecraft服务器

LongerONE 12 Oct 2016

前言

McMyAdmin是一款Minecraft服务器管理面板，功能强大，界面友好，基于 .NET 开发,适用于单个服务器管理。

本文简单介绍如何在CentOS7环境下使用McMyAdmin搭建Minecraft服务器。

步骤

安装java

  • 首先使用以下命令安装java，如已安装则跳过：

     yum install java

  • 出现如下提示则输入y后回车：

     事务概要
 ====================================================================================
 安装  1 软件包 (+14 依赖软件包)
	 
 总下载量：34 M
 安装大小：108 M
 Is this ok [y/d/N]:

  • 安装完成后验证是否安装成功：

     java -version

  • 将会显示java的版本信息：

     openjdk version "1.8.0_102"
 OpenJDK Runtime Environment (build 1.8.0_102-b14)
 OpenJDK 64-Bit Server VM (build 25.102-b14, mixed mode)

安装mono

mono可以让.NET程序在Liunx环境下运行，因mono包没有包含在CentOS的仓库里，我们需要把Mono的仓库导入到包仓库里.

  • 在root 用户权限下执行下面命令：

    rpm --import "http://keyserver.ubuntu.com/pks/lookup?op=get&search=0x3FA7E0328081BFF6A14DA29AA6A19B38D3D831EF"
yum-config-manager --add-repo http://download.mono-project.com/repo/centos/

  • 如提示-bash: yum-config-manager: 未找到命令,是因为yum-config-manager在yum-utils 包里，所以先安装：

  yum -y install yum-utils
  • 安装mono
  yum install mono
  • 验证：
  mono
  • 将会显示mono的提示信息：

#### 运行McMyAdmin

  • 从官方下载或上传McMyAdmin，推荐搭建FTP服务器并上传，同时方便后期管理文件。
  • 解压压缩包，如未压缩则跳过此步骤，例如：
  unzip MMA.zip
  • 设置管理密码:
  mono McMyAdmin.exe -setpass [YOURPASSWORD] -configonly

  • 其中[YOURPASSWORD]就是要设置的密码。此步骤会检测java是否成功安装，如失败，则尝试查看配置文件是否异常或重新安装java。

  • 运行McMyAdmin

  mono McMyAdmin.exe
  • 将会出现如下提示：
Notice  : This is the first time McMyAdmin has been started.
Notice  : You must complete the first-start wizard via the web interface.
  • 进入web管理
    打开浏览器输入VPS地址，默认端口8080,如：http://192.168.52.128:8080，然后登陆。

提示：Minecraft服务端的安装可以使用McMyAdmin下载或自行整合好服务端以后上传至Minecraft目录。

后台运行

  • 推荐使用其他用户来运行MMA，推荐使用FTP相同的账户ftpuser（需要打开shell权限）

    su ftpuser -l -c "mono McMyAdmin.exe"

进程管理

  • 使用如下指令查看进程状态:
  ps aux | grep mono
  • 将会显示McMyAdmin的进程信息,10071为进程ID：
  root      10071 66.6  6.2 1326964 63068 ?       Sl   10:11   0:22 mono McMyAdmin.exe
  • 使用如下指令关闭进程（先正常关闭服务器以防止数据丢失）:
  kill 10071

添加快速启动脚本

  • 新建一个sh脚本：
  vi start.sh
  • 然后按i进入编辑插入模式，左下角会提示INSERT,随后将启动指令输入：
  su ftpuser -l -c "mono McMyAdmin.exe"

  • 输入完成后，按Esc键退出编辑插入模式，输入:wq保存并退出。

  • 此时目录下多了一个start.sh文件，为该文件添加可以执行权限：

  chmod +x start.sh
  • 然后输入以下指令启动McMyAdmin并在后台运行:
  ./start.sh

教程完毕
