Sublime Text 3 使用技巧总结

LongerONE 07 May 2016

文本编辑作为常用的操作，当然需要一把利器。

在接触 Sublime Text 3 之前，我用的是 Notepad++ ，但在一次编辑Minecraft服务器插件 BossShop 配置时，因其换行的自动缩进导致配置文件无法被正常识别，而在软件的设置中，又难以找到相关设置以解决这个问题，于是找到了 Sublime Text 3 ，编辑后配置文件可正常使用，遂路转粉。

在这里就总结下使用的一些技巧。

下载 Sublime Text

Sublime Text官网

其他：

  • Windows版： 推荐 流风轻音
  • MAC版： Sublime Text 3107 提取码：7ql3

使用技巧

1.安装 Package Control

安装Package Control: 使用 Ctrl + ` (数字1左边的按键) 打开命令框，粘贴以下代码后回车：

import urllib.request,os; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), 'wb').write(urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace(' ','%20')).read())

重启 Sublime Text 3 后生效。

2.解决中文乱码

从菜单中打开 Tools –> Command Palette:

打开 Install Package:
未安装 Package Control 参考：1.安装 Package Control

稍等片刻后，出现 Install Package的窗口，查找 ConvertToUTF8:

选择 ConvertToUTF8 进行安装，安装完成后提示如下：

3.使用中文菜单

从菜单中打开 Tools –> Command Palette:

打开 Install Package:
未安装Package Control参考：1.安装 Package Control

稍等片刻后，出现 Install Package的窗口，查找 ChineseLocalization:

选择 ChineseLocalization 进行安装。

安装完成后，如菜单仍为英文，则 重启 Sublime Text 3 :

4.关闭更新提示

  • 英文菜单:
    打开 Sublime Text —> Preferences —> Settings - User ;
  • 中文菜单:
    打开 首选项 —> 设置 - 用户;
    输入以下代码后保存，重启 重启 Sublime Text 3 生效：
{ "update_check": false }
