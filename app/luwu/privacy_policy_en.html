LuWu Privacy Policy

Version: 2026.06.24 · Updated: 2026-06-24

App type: File manager with local encryption and vault Scope: LuWu iOS client and related backup services

1. What we collect

To run the app, we process only information that is necessary for local file management:

  • Metadata of files visible in the app: names, type, size, path, and update time.
  • In-app actions: encrypt, move to vault, restore, delete, preview, and Wi‑Fi upload status.
  • App preferences: language, theme, vault state, and encryption backup mode.
  • Wi‑Fi transfer job metadata while uploading (file name, size, progress).
The app does not collect file contents as a separate background analytics batch.

2. How we use data

  • Manage local file list, generate previews, and support extract/encrypt/decrypt operations.
  • Encrypt selected files into LWEF packages and unlock with a file password or your master password when allowed.
  • Keep sensitive files isolated in Vault when the Vault is closed.
  • Run iCloud backup/restore only after you explicitly enable the feature.

3. Storage and retention

  • File content is stored in the local app sandbox directories for your direct management.
  • Upload and temporary files are cleaned according to task lifecycle.
  • When iCloud sync is off, data is not uploaded for cloud backup.
  • You can delete local files, clear cache, or uninstall the app at any time.

4. Permissions and network

  • File access is used only to read files you choose to import or operate on, then update local lists.
  • Photo or camera permissions are requested only when the system-level operation requires it.
  • Wi‑Fi transfer opens a temporary local web endpoint in your LAN only for receiving uploaded files.

5. Third parties and sharing

LuWu does not sell, rent, or trade your personal data. No ad tracking SDK is used in this product flow.

  • The app may rely on platform services (Apple authentication, Keychain, iCloud, system file APIs).
  • Only user-enabled iCloud backup metadata/settings are synchronized.

Private file contents are not disclosed to third parties except when legally required.

6. Your controls

  • Disable or switch backup mode in Settings.
  • Delete files, clear vault items, and remove transfer logs from within the app.
  • Change language, appearance, and device permissions at any time.

For privacy questions, contact us through the App Store support channel.

7. Policy updates

When the policy changes, this page and policy version are updated. On first launch of a new version, you need to re-confirm acceptance.

Keeping this as an independent HTML file enables easier future updates.
