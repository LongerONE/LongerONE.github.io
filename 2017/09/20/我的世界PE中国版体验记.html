我的世界PE中国版体验记

LongerONE 20 Sep 2017

我的世界中国版iOS版初体验

前言

网易在去年购买了我的世界部分版权，当时就很好奇他会搞出什么花样，毕竟那时 PC java 版就形成了一定的生态，各种小游戏及模组非常的丰富。

但随着手机的性能越来越强，PE版的用户也越来越多。今年年初，时不时看到有小朋友蹲在某个角落，双手持握手机，正紧张的对抗僵尸。而 PC java 版的模组联机遭遇了大问题，Sponge 的各种频繁更新及国内的不普及，Cauldron 的种种原因，然国内的模组联机大部分都停留在了 1.7.10。

小游戏服务器生态国内不同乐观，曾有火爆一时的大型游戏服都已关闭。进入几个相对大的游戏也只有几个人，少了团队乐趣，自然就退出了游戏。

前几天发布的我的世界PE中国版在 APP Store 上架，体验一把后觉得，我的世界可能要焕发第二春了。

UI及内容

整个APP设计还是非常漂亮的。

相比官方版，中国版将各种资源进行了集中管理，皮肤、材质、地图、光影等，而新增的玩法就相当于 PC java 版的模组，相比电脑上复杂的整合方式，中国版更方便快捷的快速管理资源(竟然有我最喜欢的方块概念材质包)。

从另一方面，也避免了很多因整合导致的问题，比如崩溃，不兼容，离奇Bug等等。

这也印证了前段时间的网易盗用 Forge 代码风波。网易承认在 Forge 的基础上进行了改进。

虽然目前只有一个光影配置，而且没有水反，但在移动端已经很不错了。

联机体验

多人联机目前房间不多，随手进了一个，感觉还不错。

但联机不同于开服务器，需要依赖于房主，房主关闭游戏后所有人就掉线了。方才就莫名其妙退出了游戏，我还以为是bug。

总结

目前资源商店的内容还非常的少，但具体能发展得怎么样也不得而知，不知道网易会不会放出开发工具包让有兴趣的开发者入驻。平台已经搭建好了，后续生态发展，相信还是值得期待的。
